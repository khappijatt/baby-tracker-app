# 👶 Baby Tracker App

A comprehensive mobile parenting app for tracking everything about your little one.

Built with **Expo React Native**, **Express.js**, **PostgreSQL**, and **Firebase Auth**.

## Features
- 🍼 Feeding, diaper, sleep & medicine tracking
- 📊 Beautiful analytics with custom SVG charts  
- 💉 Vaccination & health records
- 📅 Appointments & reminders
- 📈 Growth milestones tracking
- 🔐 Firebase authentication (Email, Google, Apple)
- 👤 Per-user data isolation

## Tech Stack
- Frontend: Expo SDK 54, React Native, Expo Router, TanStack Query
- Backend: Express.js, TypeScript, Drizzle ORM
- Database: PostgreSQL
- Auth: Firebase Authentication
